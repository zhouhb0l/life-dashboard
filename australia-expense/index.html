Split Rules

Every recorded fee has its own split selector. Use the rules below to decide how each amount should be shared.

Equal split Z pays half and G pays half.
Z gets +15 `Z = (total - 15) / 2 + 15`, `G = total - Z`. For small totals below `15`, the page safely collapses this to fully Z so no negative share appears.
Fully Z / Fully G Use these for clearly personal costs or when one person should bear the whole amount.
2/3 Z Z pays two-thirds of the amount and G pays one-third.

Ledger Notes

Assumptions and conventions used in this page.

Original currencies stay separate Amounts are kept exactly as `AUD` or `SGD`, because the record is meant to stay faithful to the source notes.
Assumed currency tags If the original note omitted a currency, the page marks that assumption directly on the expense row.
Unknown payer rows If a fee did not say `@Z` or `@G`, the row starts as `TBD` and can be assigned manually later.
